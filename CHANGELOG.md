# Changelog

All notable changes to this project are documented in this file.

## [0.0.2] - 2026-06-21

### Added
- Added the first TGE resource items: dusts, fire brick, unfired clay brick, and brick mold.
- Added the reusable brick mold crafting behavior.
- Added the TGE tool framework with iron manual tools and iron vanilla-style tools.
- Added durability, remaining durability, crafting uses, and tier tooltips for TGE tools.
- Added the dedicated TGE Tools creative tab.
- Added strict `tge:tools/<tool>` tags for TGE-only tool recipe inputs.
- Added generated crafting recipes for iron manual tools, unfired clay bricks, plates, and rods.

### Removed
- Removed legacy placeholder content and old generated recipes from the previous TGE-Core setup.
